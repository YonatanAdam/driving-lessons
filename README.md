# C# MVVM SQLite Template

A lightweight MVVM template with SQLite database integration for C# desktop applications.

## Features
-  Base entity and repository pattern
-  Change tracking (Insert/Update/Delete)
-  Parameterized queries (SQL injection safe)
-  SQLite database support
-  Easy to extend with new entities

## Quick Start
1. Clone the repository
2. Add your entities (inherit from `BaseEntity`)
3. Create DB classes (inherit from `BaseDB`)
4. Add a view (e.g., WPF Window or WINUI3))
5. Start building!
